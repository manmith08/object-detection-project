# Object Detection Project

This project demonstrates object detection using Python in Google Colab.

## 📌 Features

- Detects multiple objects in an image
- Uses pre-trained models for object detection
- Displays bounding boxes with labels

## 🛠 Tools Used

- Python
- Google Colab
- OpenCV
- TensorFlow / PyTorch (depending on model used)
- Jupyter Notebook format (.ipynb)

## 🚀 How to Run

1. Open this notebook in **Google Colab**
2. Connect to a runtime
3. Run all the cells in order
4. Upload an image when prompted
5. The model will detect and display objects in the image
